# 消息面分类标准 (Message Classification Taxonomy)

## 四类标准分类

所有每日复盘/推送技能在整理消息面时，统一按以下四类分组摘要，不按时间顺序罗列：

1. **国内政策与监管** — 产业政策、监管新规、部委表态等（对应 macro-context 维度 A_政治与政策、J_资本市场制度政策）
2. **行业与产业动态** — 行业数据、产业链事件、技术突破等（对应 macro-context 维度 M_长期产业热点）
3. **公司层面** — 财报、公告、股东变动、分红回购等
4. **国际重大事件** — 央行决议、地缘政治、关税贸易等（对应 macro-context 维度 I_国际形势、K_全球主要央行政策）

## 适用范围

- `ashare-evening-review`: 白天消息面整理
- `ashare-morning-brief`: 隔夜消息汇总
- `ashare-weekly-review`: 周度热点消息汇总

## 关联标注要求

整理时标注哪些消息与已有持仓/自选股直接相关，作为观察清单的来源依据。

## 范围限定

只收录与市场整体、宏观环境或持仓/自选股所属行业相关的信息，做重点摘要，**不是所有新闻的流水账**。
