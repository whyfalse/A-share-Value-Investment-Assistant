# 早间消息推送输出模板

推送时按此结构输出,每条保持一两句话,不展开成长篇分析。盘前不跑技术引擎:外盘只做定性、技术参考位完全承接昨晚晚间复盘已算好的结果,不现算、不预测、不给买卖价位(边界见 `../../../references/boundaries.md` §7,写法见 `../../../references/technical-depth-guide.md` §4.2)。"今日观察清单"与"今日技术参考位"建议在当天保留,供14:30-14:45的 `ashare-intraday-review` 回顾衔接与对照。

```
## 早间消息推送 [日期 8:00-8:20]

### 隔夜美股
- 道指/纳指/标普收盘表现
- 中国概念股/ADR整体表现
- 美债收益率、美元指数、原油等变化
- 重要美股财报/事件关联国内板块:(如有,仅标注关联可能;否则写"无")

### 欧洲及全球其他市场
- 欧洲市场隔夜收盘表现(已是完整收盘数据)
- 隔夜全球宏观事件:(如有,否则写"无显著事件")
- (不含亚太——完整亚太数据见昨晚晚间复盘)

### 隔夜消息汇总(昨晚晚间复盘→今早)
- 国内政策与监管:[隔夜如有,否则写"无"]
- 行业与产业动态:[摘要]
- 公司层面:[摘要,标注与持仓/自选股的关联]
- 国际重大事件:[摘要]

### 今日技术参考位(承接昨晚晚间复盘,盘前不现算)
- 隔夜外盘定性:美股/欧洲/A50期货整体偏强/偏弱/分化,是否破关键位(只定性,不硬编点位、不预测A股开盘)
- 持仓今日观察基准:
  - [标的] 昨收A,昨晚测算支撑B / 压力C / 止损D(距昨收E%)→ 今日若回踩B一带可观察是否站稳(B/C/D为参考位,非下单指令)
- (无昨晚参考位时)昨晚未复盘/节后首日,无参考位 —— 仅做隔夜外盘定性,不回填、不编造

### 短期机会发掘
- [标的]: 隔夜观察到的短期催化剂/事件驱动[简述] → 为什么值得注意
- (若无)隔夜未发掘到值得注意的短期机会

### 昨晚隔夜观察事项核对
- [事项]: 隔夜进展/结论(已解释清楚/并入今日观察清单继续观察)
- (若昨晚无遗留)昨晚晚间复盘无隔夜需观察事项

### 今日已知日程
- [时间] [议题]——仅标注待观察,不预测结果

### 已完成宏观更新队列
- (如发现Tier3新事件)已产出更新建议至队列:[类别代码 + 维度] —— [简述更新内容]
- (若无)今日无需产出宏观更新建议的新事件

### 今日观察清单(衔接14:30盘中复盘)
- [标的/主题]: 关注原因(隔夜消息/昨晚遗留/今日日程/技术参考位)——仅作观察标记,非操作建议
- (若无)今日暂无需要特别观察的新增项
```

## 提醒事项

- 今日技术参考位只是把昨晚已算好的位"摆出来当今日尺子",**不现算新指标、不预测涨跌、不给买卖价位**;隔夜外盘只定性、不硬编点位(边界见 `../../../references/boundaries.md` §7)。
- 若昨晚晚间复盘报告不存在(周一/节后首日/昨晚未复盘),如实写"无昨晚参考位",仅做隔夜外盘定性,不回填、不编造历史测算位。
- 若隔夜出现可能影响某中长期持仓的重大消息,在输出中明确写"已标记观察,按既有止盈止损/纪律线处理,不建议开盘前临时操作",而不是直接给出操作倾向。
- 今日观察清单中的条目,到14:30盘中复盘时应有结论(已解释清楚/仍在观察/已转入宏观记忆),不要每天累积新条目却从不清空。
- 隔夜消息摘要聚焦"与市场、宏观环境或持仓/自选股相关行业有关"的信息,不做泛新闻聚合;范围把握不准时,优先保留与持仓/自选股直接相关的条目。
- 本skill不涉及亚太市场(日韩港),亚太由昨晚 `ashare-evening-review` 以完整收盘数据覆盖,避免重复与时段错位。
