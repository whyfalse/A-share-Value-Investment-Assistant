# 晚间复盘输出模板

复盘时按此结构输出,每条保持一两句话,不展开成长篇分析。技术面只做"客观状态描述 + 纪律线校准 + 对称条件式情景",不预测涨跌、不给买卖价位(边界见 `../../../references/boundaries.md` §7,写法见 `../../../references/technical-depth-guide.md`)。"隔夜需观察事项"与"技术参考位"建议在当晚保留,供次日 `ashare-morning-brief` 核对隔夜进展与承接观察基准。

```
## 晚间复盘 [日期]

### 今日A股全天回顾
- 指数表现:上证XX 深证XX 创业板XX(涨跌幅)
- 资金流向:北向/南向资金当日净流向
- 领涨/领跌板块:[简述]
- 涨跌停家数等情绪指标:[简述]
- 大盘技术面(承接技术引擎结果):
  | 指数 | 收盘 | 均线排列 | vs MA20/60/250 | ADX | 状态 |
  |---|---|---|---|---|---|
  | 上证 | XX | 多头/空头/纠缠 | 上/下 | XX | 健康上行/高位震荡/趋势走弱/底部修复 |
  | 深证成指 | … | … | … | … | … |
  | 创业板指 | … | … | … | … | … |
  | 科创50 | … | … | … | … | … |
  → 风格分化:权重蓝筹 vs 成长谁强谁弱;一条对称条件式情景(以关键均线/整数关口为锚)
- 持仓与自选股相关提示:(如有,与盘中复盘衔接不重复;否则写"无")

### 亚太市场(辅助)
- 日经/恒指/韩股(KOSPI)当日收盘表现——同交易时段,已是完整收盘数据
- 与A股的联动/背离:(仅客观陈述,不预测次日;否则写"无显著背离")

### 白天消息面(今晨早间消息推送→今晚)
- 国内政策与监管:[摘要]
- 行业与产业动态:[摘要]
- 公司层面:[摘要,标注与持仓/自选股的关联]
- 国际重大事件:[白天如有,否则写"无"]

### 板块轮动持续性
- 轮动方向:今日强势板块相对昨日/本周是延续还是切换
- 领涨板块持续性定性:量价资金是否共振(放量+净流入+龙头连板=强;缩量+流出+冲高回落=一日游嫌疑)
- 持仓板块定位:处在轮动的主线/补涨/退潮位置,个股顺风还是逆风

### 持仓/自选股四维深度卡片(逐只,中长期与短期口径分栏不混用)
#### [代码 名称] ｜ 今日 ±X.XX%(板块 ±Y%,沪深300 ±Z%)
- 技术面:[均线排列 / MACD / KDJ / RSI / bias20 / ADX] → 客观状态描述(如"趋势健康上行、短期略过热")
- 资金面:主力近X日净流入/流出 … ;OBV 20日趋势 上升/走平/下降
- 消息面:[仅与该股/行业相关的重点;无则"今日无直接相关消息"]
- 相对强弱:alpha_20d=±X%、alpha_60d=±X% → 今日异动是个股alpha / 跟随大盘beta(基准缺失则写"本次缺相对强弱")
- 条件式情景(对称):若放量站稳 MAxx(约A)则上行延续确认;若跌破 MAyy 且 MACD死叉则警惕回调。(A为参考位,非下单指令)
- 纪律校准:现价P,距止盈XX(+M%)、止损XX(−N%)、加仓线(若有);当前均未触及/已逼近XX线,请核对自己的纪律
  (未满最低持有5日的持仓:即便出空头信号也只"标记观察/核对止损线",不建议提前卖出)

### 短期机会发掘
- [标的]: 今日全天观察到的短期催化剂/异动[简述] → 为什么值得注意
- (若无)今日未发掘到值得注意的短期机会

### 盘中未结事项核对
- [事项]: 结论(已解释清楚/仍在观察/已转入宏观记忆)
- (若盘中无遗留)今日盘中复盘无未结事项

### 已完成宏观更新队列
- (如发现Tier3新事件)已产出更新建议至队列:[类别代码 + 维度] —— [简述更新内容]
- (若无)今日无需产出宏观更新建议的新事件

### 隔夜需观察事项(交棒明早早间消息推送)
- [标的/主题]: 关注原因(来自白天消息或盘中遗留)——仅作观察标记,非操作建议
- 技术参考位移交:各持仓支撑/压力/纪律线 —— 供次日早间/盘中作为观察基准
- (若无)今晚暂无需要隔夜跟进的事项
```

## 提醒事项

- 技术面三条铁律(详见 `../../../references/boundaries.md` §7):只做客观状态描述+纪律线校准;条件式情景必须**两个方向都写**,禁止"明日大概率/上涨概率XX%"的单边预测;支撑压力/止盈止损都是**参考区间非下单指令**,绝不写成"明日XX.XX元买入/卖出"。
- 技术引擎某维度因数据不足/基准缺失被跳过时,如实写"该维度缺失",**不编造指标数字**填报告(降级规则见 `technical-depth-guide.md` §七)。
- 若白天出现可能影响某中长期持仓的重大消息,在输出中明确写"已标记观察,按既有止盈止损/纪律线处理,不建议盘后临时操作",而不是直接给出操作倾向。
- 盘中未结事项要在晚间被消化(给出结论或转为隔夜观察),不要让同一疑点在盘中→晚间→早间之间无限循环却始终没有结论。
- 白天消息摘要聚焦"与市场、宏观环境或持仓/自选股相关行业有关"的信息,不做泛新闻聚合;范围把握不准时,优先保留与持仓/自选股直接相关的条目。
- 亚太市场只在本skill出现(数据完整);海外完整隔夜行情(美股/欧洲)交给次日早间消息推送,本skill不展开。
