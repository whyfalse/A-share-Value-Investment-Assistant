# 盘中复盘输出模板

复盘时按此结构输出,每条保持一两句话,不展开成长篇分析。

```
## 盘中复盘 [日期 14:30-14:45]

### 持仓异动 / 纪律核对
- [标的]: 现象 → 是否触及预设线(止盈/止损/加仓) → 建议关注/无需处理
- (若全部正常)今日持仓无显著异动,均未触及预设纪律线

### 自选股提醒
- [标的]: 是否触及预设买点条件 / 是否异动
- (若无)今日自选股无触发提醒

### 宏观背景校准
- 今日现象 ↔ 对应的已知宏观背景(引用ashare-macro-context记忆文档中的具体维度)
- (若有新事件且记忆文档未覆盖)发现新事件:[简述],已标记待补充进宏观记忆,初步判断可能涉及:[相关持仓/自选股],仅作观察标记,不构成操作建议

### 大盘/板块温度计
- 一句话概括今日指数及相关板块表现,用于区分系统性波动与个股信号

### 短期机会状态
- [标的]: 催化剂[简述] → 状态:pending/confirmed/invalidated

### 待办
- 今天剩余交易时间/收盘后需要处理:[事项]
- 留待晚间/周末完整复盘的疑点:[事项]
```

## 提醒事项(每次复盘结尾视情况附加)

- 若复盘中用户表现出想用当日消息推翻中长期持仓逻辑的倾向,温和提醒最低持有5个交易日的纪律,不直接配合给出"卖出"建议。
- 若某条短期机会的判断依据开始向"基本面叙事"靠拢,提醒用户这条应该转入持仓/自选股体系,用价值投资的标准重新评估,而不是继续用短期催化剂框架硬撑。
