# 盘中复盘输出模板

复盘时按此结构输出,每条保持一两句话,不展开成长篇分析。技术位温度计只对照昨晚晚间复盘已算好的参考位,盘中不现算指标、不预测方向、不给买卖价位(边界见 `../../../references/boundaries.md` §7,写法见 `../../../references/technical-depth-guide.md` §4.3)。

```
## 盘中复盘 [日期 14:30-14:45]

### 持仓异动 / 纪律核对
- [标的]: 现象 → 是否触及预设线(止盈/止损/加仓) → 建议关注/无需处理
- (若全部正常)今日持仓无显著异动,均未触及预设纪律线

### 技术位互动温度计(对照昨晚晚间复盘参考位,盘中不现算)
| 持仓 | 现价 | vs 昨晚支撑 | vs 昨晚压力 | 距止盈/止损 | 盘中量价 |
|---|---|---|---|---|---|
| [标的] | XX | 上方+X%/下方−X% | 逼近/已破/距+X% | 距止损−X%(触及才提示) | 放量/缩量配合方向 |
- (无昨晚参考位时)昨晚未复盘/节后首日,无参考位 —— 仅做盘中量价定性,不回填、不编造
- (未满最低持有5日的持仓)即便跌破昨晚支撑也只"标记观察/核对止损线",不建议卖出

### 自选股提醒
- [标的]: 是否触及预设买点条件 / 是否异动
- (若无)今日自选股无触发提醒

### 宏观背景校准
- 今日现象 ↔ 对应的已知宏观背景(引用ashare-macro-context记忆文档中的具体维度)
- (若有新事件且记忆文档未覆盖)发现新事件:[简述],已产出更新建议至队列,初步判断可能涉及:[相关持仓/自选股],仅作观察标记,不构成操作建议

### 大盘/资金流向/板块温度计
- 指数表现与涨跌停/涨跌家数情绪,一句话区分系统性波动与个股信号
- 实时资金流向:主力/北向净流入流出总量 → 主要流入板块:[...]、主要流出板块:[...]
- 主要个股涨跌:涨跌幅榜前列或显著放量异动个股,标注是否与持仓/自选股相关

### 短期机会发掘
- [标的]: 今日观察到的短期催化剂/异动[简述] → 为什么值得注意
- (若无)今日未发掘到值得注意的短期机会

### 待办
- 今天剩余交易时间/收盘后需要处理:[事项]
- 留待晚间/周末完整复盘的疑点:[事项]
```

## 提醒事项(每次复盘结尾视情况附加)

- 技术位温度计只是"现价 vs 昨晚位"的客观距离描述,不预测方向、不给买卖价位;纪律线只在**触及时**提示"请核对是否执行你自己的纪律",绝不替用户下"该买/该卖"结论。
- 若复盘中用户表现出想用当日消息推翻中长期持仓逻辑的倾向,温和提醒最低持有5个交易日的纪律,不直接配合给出"卖出"建议。
- 若某条短期机会的判断依据开始向"基本面叙事"靠拢,提醒用户这条应该转入持仓/自选股体系,用价值投资的标准重新评估,而不是继续用短期催化剂框架硬撑。
