# A股价值投资助手

## 概述
### 功能
针对A股的价值投资AI助手，根据用户的个人投资偏好（持仓股、自选股、关注板块等信息），为用户进行投资辅助。

### 限制

- 只适用于A股的中长期投资（至少持股5个交易日）。
- 不会对用户的交易记录进行分析。
- 不会进行买卖建议。
