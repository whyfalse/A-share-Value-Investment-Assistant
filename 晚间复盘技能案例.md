# 晚间复盘（Evening Review）

收盘后（15:00后）全面深度总结工具。基于AKShare收盘数据+技术指标计算+联网搜索消息面，输出包含明日操作计划的全面复盘报告。

## 前置条件

- AKShare 及依赖已安装：`pip install akshare numpy pandas`
- 持仓文件已配置：`config/portfolio.json`
- 建议在15:00收盘后运行以获取完整日数据

## 分析流程

按以下步骤依次执行。晚间复盘数据量大，每完成一个步骤汇报简要进度。

### 第一步：读取用户持仓

1. 使用 Read 工具读取持仓文件。如果用户通过 `$ARGUMENTS` 指定了路径则使用该路径，否则默认读取 `config/portfolio.json`
2. 确认持仓列表，向用户展示："晚间复盘将深度分析以下XX只持仓股：XX股、YY股…"
3. 确认用户是否有特别关注的方面（可选跳过直接进入全量分析）

### 第二步：获取收盘数据与技术指标

运行晚间数据获取脚本：

```bash
cd d:/ai-design/ai-market-review && python .claude/skills/evening-review/scripts/evening_data.py --portfolio config/portfolio.json
```

> **注意**：晚间数据脚本会计算技术指标（MA/MACD/KDJ/RSI/BOLL），需要更长的执行时间（通常10-30秒）。请等待脚本完全执行完毕。

脚本返回数据维度（详细字段见脚本输出）：

**指数日线分析（index_daily_analysis）**：
- 四大指数最近60个交易日数据
- 各指数MA5/MA10/MA20/MA60均线值
- 成交量分析（当日成交量 vs 5日/20日均量）
- 涨跌幅

**市场情绪全貌（market_breadth_detail）**：
- 涨跌家数分布（含+5%以上暴涨数和-5%以上暴跌数）
- 涨停/跌停家数
- 市场平均涨幅、中位数涨幅（赚钱效应指标）
- 涨幅分段统计

**两市成交量（market_volume）**：
- 上证成交额、深证成交额
- 成交量与近期平均水平对比

**北向/南向资金（north_south_flow）**：
- 北向资金近20日净流入/流出趋势
- 南向资金近20日流向
- 沪深港通十大成交股（如有）

**板块全排名（sector_full_ranking）**：
- 所有行业板块涨跌幅完整排名
- 板块资金流向排名

**主力资金（main_fund_flow）**：
- 两市主力资金净流入/流出前十

**涨停板详情（limit_up_detail）**：
- 涨停股数量及名单
- 连板天数统计

**持仓个股深度分析（portfolio_deep_analysis）**：
每只持仓股包含：
- 最近20个交易日K线数据（OHLCV）
- 技术指标计算结果：MA5/MA10/MA20/MA60、MACD(DIF/DEA/MACD柱)、KDJ(K/D/J)、RSI14、BOLL(上轨/中轨/下轨)
- 量价分析：当日成交量、5日/20日均量、量比
- 关键价位：60日支撑位/压力位、最新收盘价
- 收盘行情数据：涨跌幅、换手率、市盈率、市净率
- 盈亏状态计算
- 个股资金流向近10日数据

### 第三步：联网搜索深度消息

使用 WebSearch 工具进行**多轮全面搜索**：

**第一轮：当日重大政策与宏观消息**
```
搜索词：今日 央行 货币政策 证监会 重要政策 A股
搜索词：今日 宏观经济数据 CPI PMI GDP 公布
```

**第二轮：隔夜及当日国际市场**
```
搜索词：美股 道指 纳指 收盘 2026年6月
搜索词：今日 亚太股市 日经 恒生 收盘
搜索词：A50 期货 人民币 汇率 今日
```

**第三轮：持仓板块行业动态**（对每个关注板块）
```
搜索词：<板块名称> 行业政策 新闻 <当前日期>
搜索词：<板块名称> 龙头股 研报 评级
```

**第四轮：持仓个股消息**（对每只持仓股）
```
搜索词：<股票名称> 公告 业绩 新闻
搜索词：<股票名称> 研报 目标价 评级
搜索词：<股票名称> 大宗交易 增减持
```

**第五轮：明日潜在热点**
```
搜索词：明日 A股 热点 板块 预测
搜索词：近期 题材 概念 炒作 A股
```

### 第四步：综合分析生成晚间复盘报告

基于以上全部数据，生成完整的晚间复盘报告。这是核心输出，务必深入、具体、可执行。

## 复盘后的可选操作

报告生成完毕后，主动询问用户：

1. **是否需要调整持仓配置？** — 可帮助编辑 `config/portfolio.json` 添加/删除持仓股、修改成本或止损止盈价位
2. **是否需要回测分析？** — 可以对持仓股在类似历史市场环境下的表现进行回测
3. **是否需要生成可视化图表？** — 可以生成K线图标注关键价位、板块热力图等（需要额外安装 matplotlib）
4. **是否需要保存为PDF？** — 可以将Markdown报告转换为PDF

## 异常处理

- **AKShare接口部分失败**：标注失败项，其他数据正常分析。晚间复盘数据维度多，允许部分缺失
- **某只持仓股K线数据获取失败**：跳过该股的技术分析，标注"数据不可用"，但通过WebSearch补充消息面信息
- **搜索结果不相关**：标注消息来源不充分，提示用户手动补充重要消息
- **非交易日运行**：提醒用户当天为周末/节假日，使用最近一个交易日数据进行分析
